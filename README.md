

# austlogin

### 简介

用 `python3` 做的命令行程序，用于登录安理大校园网

使用 `pyinstaller` 打包exe

### 使用方法

将exe文件添加至环境变量，然后 `austlogin -h` 查看帮助

或 `python main.py -h` 查看帮助

详细教程: [安理工校园网快捷登录的超详细教程(安卓/PC端) · 心海/博客 - 语雀](https://www.yuque.com/xinhai1/ukxtbt/zpb8yz)
